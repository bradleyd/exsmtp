Exsmtp
======

Basic SMTP server for Elixir. 
It uses the wonderful [gen_smtp](https://github.com/Vagabond/gen_smtp)

There was also code inspired by [HashNuke's Mail To JSON](https://github.com/HashNuke/mail-to-json)
